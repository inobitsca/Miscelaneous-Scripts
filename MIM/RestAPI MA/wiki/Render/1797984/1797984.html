Project Description

Generic REST MA is Management Agent for Forefront Identity Manager 2010 R2 to synchronize objects between FIM and SaaS applications using RESTful APIs.

 

Features

  • Plugin architecture. By creating a plugin library, you can extend connectivity with new SaaS application.
  • Support connection using proxy server with authentication.
  • Include sample plugin library for Google Apps, so you can synchronize objects with Google Apps by default.

 

 

Prerequirement

  • Target SaaS application
    • Support OAuth2.0 JWT Bearer Token Flow for authorization.
    • Support RESTful API for object management.
    • Using JSON to represent objects.
  • Forefront Identity Manager environment
    • ECMA2.3 (Forefront Identity Manager 2010 R2 SP1 build 4.1.3559.0 or later)
    • .NET Framework 4.5
